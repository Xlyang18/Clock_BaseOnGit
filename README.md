# 任务打卡管理工具

## 介绍 

仿制Git分布式管理工具开发的任务打卡管理工具

该工具允许根据支持的简单指令对后台数据库直接进行操作，记录下打卡任务项目名以及操作时间，并对每一条记录生成一个10位数的随机码标记。

# 现支持的指令

- 添加新的记录—> add 【-记录名称】
- 查看最近记录—>  log  *
- 查看全部记录—>   all   *
